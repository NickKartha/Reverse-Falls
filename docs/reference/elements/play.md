
# [command] play

Starts playing an audio track from an [audio](audio.md) asset.

## Usage

    . play @my_audio

## Attributes

 * **asset/@[asset_name]:** The audio asset's internal name.
 * **ifvar:** See [conditionals](conditionals.md).
