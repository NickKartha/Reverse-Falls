
# [element] audio/track

Describes one track of an [audio](audio.md) asset. It needs to contain at least one
[source](source.md) element for the track to work.


## Usage

See [audio](audio.md).


## Attributes

 * **title:** The track's title by which it can be referenced on the `<audio>` asset.

## Children
